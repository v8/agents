# Gemini Extensions

This directory contains extensions / MCP ([model context protocol]) server
configurations useful for V8 development. Each subdirectory within this
directory corresponds to one extension.

Configuration are provided in [gemini-cli extensions] format.

## Creating an MCP server

See the [example][3] server for a minimal example for creating an MCP tool with
python and FastMCP
